This project holds language files for the framework.
At present it contains only english version jriapp_en.ts.
The language javascript file (for example jriapp_en.js) must be added to html page scripts before main jriapp.js file.
For example:
    <script src="/Scripts/jriapp/jriapp_en.js" type="text/javascript"></script>
    <script src="/Scripts/jriapp/jriapp.js" type="text/javascript"></script>
